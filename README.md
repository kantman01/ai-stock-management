# AI Stock Management System

A modern stock management system built with React and Node.js, featuring AI-powered analytics and predictions.

## Features

- Product Management
- Category Management
- Order Processing
- Customer Management
- Supplier Management
- Stock Movement Tracking
- AI Analytics Dashboard
- User Authentication & Authorization
- Role-based Access Control

## Tech Stack

### Frontend
- React
- Material-UI
- Redux Toolkit
- React Router
- Axios
- Chart.js

### Backend
- Node.js
- Express
- PostgreSQL
- JWT Authentication
- OpenAI Integration
